Dentist Website Redesign

This site was my first project at The Iron Yard, and the assignment was to redesign the website for a local dentist office. We started withresearch, content strategy, branding, competitive analysis, and then implemented the elements of design we had learned and then coded the site using the "design in the browser" method.

I learned a lot about content strategy through this project and I believe the finished product successfully reflects the mission of the dentistry


Technologies Used:

  • CSS (Sass)
